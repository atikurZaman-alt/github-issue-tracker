1️⃣ What is the difference between var, let, and const?
answer :- var,let, and const are all used to declare variables in javascript but they behave differently. var is the older way and it is function-scoped which means it can be accessed throughout the whole function and can also be redeclared or updated easily.let is block-scoped so it only works inside the block{} where it is created and it can be updated but not redeclared in the same scope.
const is also block-scoped like let but its value cannot be reassigned after it is declared which makes it useful for values that should stay constant.

2️⃣ What is the spread operator (...)?
answer:- the spred operator (...) is used to expand elements of an array or properties of an object. it helps copy,merge, or pass values easily in a cleaner way.

3️⃣ What is the difference between map(), filter(), and forEach()?
answer:- map(),filter(), and forEach() are used to work with arrays in javaScript. map() creates a new array by changing each element, filter() creates a new array with only the elements that match a condition, and forEach() simply runs a function on each element without returning a new array.

4️⃣ What is an arrow function?
answer:- an arrow function is a shorter and simpler way to write a function in javaScript using the => symbol. it helps make the code cleaner and easier to read especially when writing small functions.

5️⃣ What are template literals?
answer :- template literals are a way to write strings in javaScript using backticks (``). They make it easier to add variables or expressions inside a string using ${} and help write cleaner more readable code.
